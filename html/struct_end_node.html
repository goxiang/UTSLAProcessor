叶子节点是四叉树的终端节点,直接存储点信息:

  1. 更新节点的边界范围
  2. 增加节点计数
  3. 将点添加到点集合中
  4. 设置点所属的节点

参数
pt要移除的点信息指针

叶子节点的点移除流程:

  1. 检查点是否在当前节点的点集合中
  2. 如果存在父节点,递归调用父节点的removePt
  3. 减少节点计数
  4. 从点集合中移除该点
